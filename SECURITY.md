# Security policy

busTracker is currently a planning-stage public repository and has no supported production release.

Please do not open public issues containing credentials, private operator feeds, customer information, exploit details, or security reports. Report sensitive findings privately through the repository owner's GitHub profile until a dedicated security-reporting address is published.

The repository must never contain `.env`, `private.md`, raw private feed captures, contracts, customer data, production configuration, signing material, or penetration-test reports.
